# SU-Enrollment-Bot
บอทช่วยล็อกอินและกดลงทะเบียนเรียน ม.ศิลปากร พัฒนาด้วย Python + Selenium (Web Automation)
---

## 📌 จุดประสงค์ของโปรแกรม (Project Purpose)
โปรแกรมนี้ถูกพัฒนาขึ้นมาเพื่อเป็น **"โปรเจกต์ทดลองเล่น ๆ"** สำหรับศึกษากระบวนการทำงานของระบบ Web Automation ด้วย Selenium เท่านั้น ผู้พัฒนาไม่ได้มีเจตนาทุจริต หรือทำลายระบบใด ๆ ทั้งสิ้น 

## 🚨 DISCLAIMER (คำปฏิเสธความรับผิดชอบ)
* ผู้พัฒนาจะไม่รับผิดชอบต่อความเสียหายใด ๆ ที่เกิดขึ้นจากการนำบอทนี้ไปใช้งานในทุกกรณี! (ไม่ว่าจะเป็นการลงทะเบียนผิดพลาด, วิชาหาย, ระบบล็อก หรือถูกลงโทษตามกฎของมหาวิทยาลัย)
* **ผู้ใช้งานต้องยอมรับความเสี่ยงด้วยตัวเองทั้งหมด**

## 🛑 คำเตือนสำคัญจากประสบการณ์ตรงของผู้สร้าง
บอทตัวนี้ทำผู้พัฒนาลงทะเบียนเรียนผิดพลาดมาแล้ว เพราะดันเจอบั๊กระบบเว็บแจ้งเตือนว่า **"การลงทะเบียนไมู่ยู่ในเวลา"** ทั้ง ๆ ที่ถึงเวลาลงทะเบียนแล้ว (08:30 น.) 

* **สาเหตุ:** น่าจะเพราะในlistดันมีวิชาพิเศษบางวิชา (เช่น `SU402`) ที่ล็อกเงื่อนไขไว้เฉพาะคณะอื่น ระบบเว็บเลยบล็อกไม่ให้กดตรวจสอบ
* ตอนนั้นผมเองกดถอนวิชาเจ้าปัญหาออก และถอนวิชาที่อยากเรียนออกไปตัวหนึ่งเพื่อลองระบบใหม่... แต่ดัน **"ลืมปิดบอท"** ผลคือบอทที่รันค้างอยู่มันทำงานต่ออัตโนมัติและกดล็อกผลทันที! สรุปคือวิชาที่อยากเรียนจริงๆ ก็ไม่ได้ลง แถมต้องไปรอเพิ่ม-ถอนวิชาที่ตัวเองไม่อยากเรียนแทน 

### 💡(RECOMMENDATION)
เลือกเฉพาะวิชาที่เสี่ยงเต็มไว้ ส่วนวิชาอื่นๆที่เป็นของวิชาทั่วไปที่โควตาเยอะ หรือวิชาที่ทุกคณะมาเรียนรวมกัน ค่อยมาเลือกช่วงเพิ่มถอน **(วิชาที่ขึ้นต้นด้วยSUต้องเช็คดีๆว่าคณะเราสามารถลงได้ไหม ไม่งั้นจะเจ๊งแบบผม)**

## ✨ ฟีเจอร์เด่นของบอทตัวนี้ (Features)
* ⚡ **Speed Optimization:** ปิดการโหลดรูปภาพและปิดการแจ้งเตือนของ Chrome เพื่อรีดความเร็วในการโหลดหน้าเว็บขั้นสุด
* ⏰ **Scheduled Run:** ระบบตั้งเวลาสแตนด์บายทำงานล่วงหน้า สามารถตั้งเวลาให้บอทเริ่มรันอัตโนมัติ (เช่น ตั้งไว้ที่ 08:29:55 น.) เพื่อให้ล็อกอินเข้าไปรอสับคอมโบปุ่มทันทีที่ระบบเปิด
